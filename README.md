[Michael Steinbaugh](http://mike.steinbaugh.com)<br />

Common R functions for use with RNA-seq, ChIP-seq, and microarrays.

To install, run `devtools::install_github("seqcloud/seqcloudR")` in [R](https://www.r-project.org).

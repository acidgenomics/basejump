## Installation

This is an [R][] package.

### [Bioconductor][] method

```r
source("https://bioconductor.org/biocLite.R")
biocLite("devtools")
biocLite("steinbaugh/basejump")
```

### [conda][]  method

```bash
conda install -c bioconda r-basejump 
```

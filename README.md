## Installation

This is an [R][] package.

### [Bioconductor][] method

```{r}
source("https://bioconductor.org/biocLite.R")
biocLite("steinbaugh/basejump")
```

### [`devtools`][] method

```{r}
install.packages("devtools")
devtools::install_github("steinbaugh/basejump")
```
